# 🔌 물리 계층 (Physical Layer)

물리 계층은 OSI 모델의 첫 번째 계층으로, 네트워크의 물리적 연결과 신호의 전송을 담당합니다. 이 계층은 전기적, 기계적, 절차적 특성들을 정의합니다.

## 📚 기본 개념

**물리 계층의 역할**:
- 물리적인 매체를 통해 데이터를 전송하는 것을 담당합니다.
- 비트 스트림을 전기적 또는 광학적 신호로 변환하여 전송합니다.
- 네트워크 장비 간의 실제 연결을 관리합니다.

## 🛠️ 주요 기능

### 1. 신호 전송 (Signal Transmission)

**아날로그 신호 (Analog Signal)**:
- 연속적인 값으로 표현되는 신호.
- 주로 음성이나 영상의 전송에 사용됩니다.

**디지털 신호 (Digital Signal)**:
- 이산적인 값으로 표현되는 신호.
- 데이터 통신에서 주로 사용됩니다.

### 2. 인코딩 기법 (Encoding Techniques)

**NRZ (Non-Return to Zero)**:
- 0과 1을 서로 다른 전압 수준으로 표현하는 가장 단순한 인코딩 방식.

**NRZI (Non-Return to Zero Inverted)**:
- 1을 전압의 변화로, 0을 전압의 유지로 표현.

**Manchester Encoding**:
- 각 비트를 중간에서 전압 변화를 사용하여 표현.
- 높은 신뢰성과 동기화 용이성을 제공.

### 3. 전송 매체 (Transmission Medium)

**유도 매체 (Guided Medium)**:
- **꼬임쌍선 (Twisted Pair)**:
  - 두 가닥의 절연된 구리선을 꼬아 만든 케이블.
  - 전화선이나 LAN에 사용.
  - UTP (Unshielded Twisted Pair)와 STP (Shielded Twisted Pair)로 구분.

- **동축 케이블 (Coaxial Cable)**:
  - 내부 도체를 절연체와 외부 도체가 감싸고 있는 구조.
  - TV 케이블이나 장거리 전화선에 사용.

- **광섬유 (Optical Fiber)**:
  - 빛의 전반사를 이용하여 데이터를 전송.
  - 장거리 데이터 전송이나 고속 네트워크에 사용.

**비유도 매체 (Unguided Medium)**:
- **무선 전파 (Radio Waves)**:
  - 넓은 지역에 걸쳐 데이터를 전송.
  - WiFi, 블루투스, 셀룰러 네트워크에 사용.

- **마이크로파 (Microwave)**:
  - 지향성 전송을 통해 데이터를 전송.
  - 위성 통신이나 점대점 통신에 사용.

- **적외선 (Infrared)**:
  - 근거리 무선 통신에 사용.
  - 리모컨이나 근거리 데이터 전송에 사용.

### 4. 전송 모드 (Transmission Mode)

**단방향 (Simplex)**:
- 데이터가 한 방향으로만 전송되는 모드.
- 예: TV 방송

**반이중 (Half-Duplex)**:
- 양방향 전송이 가능하지만, 한 번에 한 방향으로만 전송.
- 예: 무전기

**전이중 (Full-Duplex)**:
- 양방향으로 동시에 데이터 전송이 가능.
- 예: 전화 통화

### 5. 전송 속도와 대역폭 (Transmission Speed and Bandwidth)

**전송 속도 (Data Rate)**:
- 단위 시간당 전송되는 비트의 수.
- bps (bits per second)로 측정.

**대역폭 (Bandwidth)**:
- 주파수 범위로 측정되는 매체의 용량.
- Hz로 측정.

### 6. 신호 감쇠와 왜곡 (Signal Attenuation and Distortion)

**감쇠 (Attenuation)**:
- 신호가 전송 매체를 통해 전달되면서 약해지는 현상.
- 증폭기나 리피터를 사용하여 보상 가능.

**지연 왜곡 (Delay Distortion)**:
- 신호의 주파수 성분들이 서로 다른 속도로 전파되어 발생하는 왜곡.
- 디지털 신호 전송에서 문제가 될 수 있음.

**잡음 (Noise)**:
- 전송 매체에서 발생하는 불규칙한 신호.
- 열잡음, 누화잡음, 충격잡음 등이 있음.

## 🌟 물리 계층의 장단점

**장점**:
- 데이터의 실제 전송을 담당하여 네트워크의 기본적인 기능을 제공.
- 다양한 매체와 인코딩 기법을 통해 유연한 데이터 전송 가능.

**단점**:
- 신호 감쇠와 잡음 등의 문제로 인해 신뢰성 있는 전송이 어려울 수 있음.
- 매체와 기법에 따라 전송 거리가 제한될 수 있음.

물리 계층은 네트워크 통신의 가장 기초적인 부분으로, 다양한 전송 매체와 기법을 통해 데이터를 전송합니다.   
이를 통해 상위 계층들이 데이터를 안정적으로 송수신할 수 있도록 지원합니다.
